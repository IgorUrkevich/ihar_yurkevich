# Язык разметки Markdown
## Заголовки
``` fix
 # Заголовок первого уровня #
 ## Заголовок второго уровня ##
 ### Заголовок третьего уровня ###
 #### Заголовок четвёртого уровня ####
 ##### Заголовок пятого уровня #####
 ###### Заголовок шестого уровня ######
```
**Пример:**
# Заголовок первого уровня
## Заголовок второго уровня
### Заголовок третьего уровня
#### Заголовок четвёртого уровня
##### Заголовок пятого уровня
###### Заголовок шестого уровня
## Выделение текста
Выделяется с обоих сторон
``` fix
*курсив*  
_курсив_

**жирный**  
__жирный__

***жирный курсив***  
___жирный курсив___

~~зачеркнутый~~
```
**Пример:**
*курсив*  
_курсив_
**жирный**  
__жирный__
***жирный курсив***  
___жирный курсив___
~~зачеркнутый~~
